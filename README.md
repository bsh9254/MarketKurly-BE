## 🏹 Trouble Shooting
- Swagger를 설정했으나 접근이 안되는 문제가 발생
=> addResourceHandlers를 오버라이딩 해서 해결함

- 컬럼명이 카멜표기법과 스네이크표기법으로 서로 다른부분이 있어서 컬럼명을 찾을 수 없는 문제가 발생
=> 카멜표기법으로 통일하여 컬럼명을 작성해서 해결함
